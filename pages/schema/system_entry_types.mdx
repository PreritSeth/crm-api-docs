# System Entry Types

All entry types can be considered type Entity. However certain entry type provided additional capabilities and workflows. The standard entry types can also be used as a template when generating new entry types, as each entry type has default fields. With the exception of the name field all other fields can be deleted, so it is always best to look at the fields of a given entry type to confirm the actual attributes.
