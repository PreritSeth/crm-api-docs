# System Field Types

Returns all system field types. System field types are a super set of field types with the guarantee that users cannot modify their names. You are also not able to write data into system fields.

## Swagger
![System Field Types](./system_field_types.png)
## Params

None

## Request
```
GET {{host}}/api/rest/v4/schema/systemfieldtypes
Authorization: {{auth}}
```
