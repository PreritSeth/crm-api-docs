# Users

Returns users or active users. Useful when you need to update/modify a user reference field.

## Swagger
![Swagger](./users.png)


## Params

| name       | type    | values       |
| ---------- | ------- | ------------ |
| activeOnly | boolean | true / false |

## Request

**Active User**

```
GET {{host}}/api/rest/v4/schema/users
    ?activeOnly=true
Authorization: {{auth}}
```

**All Users**

```
GET {{host}}/api/rest/v4/schema/users
Authorization: {{auth}}
```
